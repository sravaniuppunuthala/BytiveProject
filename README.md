# Todo App Assignment

This is a backend application for managing a Todo list, built as part of an assignment. 
The application provides RESTful APIs to perform CRUD operations on tasks, such as creating, updating, viewing, and deleting todos.

## Features

- Create a new todo task
- Retrieve a list of all tasks
- Update existing tasks
- Delete tasks
- Mark tasks as completed
- Backend implemented with Spring Boot
- Database integration with MySQL
- Secure authentication using JWT

---

## Technologies Used

- **Java**: Backend language
- **Spring Boot**: Framework for RESTful APIs
- **MySQL**: Database for storing todo data
- **JWT (JSON Web Token)**: Authentication and authorization
- **Maven**: Build automation tool
- **Postman**: For testing APIs
